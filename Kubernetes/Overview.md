Azure Kubernetes (K8s)
• Kubernetes is an open-source container orchestration platform.
• It automates the deployment, scaling, and management of containerized applications.
• Originally developed by Google, now maintained by the Cloud Native Computing Foundation (CNCF).
• Offers features like load balancing, self-healing, and automated rollouts/rollbacks.
• Supports various deployment environments, including onpremises, public, and hybrid clouds.

Azure Kubernetes (K8s)
• Pods: Unit of deployment in Kubernetes
• Deployments: Manages the lifecycle of pods
• Services: Provides a stable endpoint to access a set of pods
• Nodes: The underlying machines where containers run
